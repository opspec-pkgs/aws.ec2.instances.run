# Problem statement

Runs one or more aws ec2 instances

# Example usage

## Visualize

```shell
opctl ui github.com/opspec-pkgs/aws.ec2.instances.run#1.1.0
```

## Run

```
opctl run github.com/opspec-pkgs/aws.ec2.instances.run#1.1.0
```
